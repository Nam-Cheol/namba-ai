# NambaAI v0.4.1

이번 릴리스는 안정성과 유지보수성을 다듬는 패치 릴리스입니다.

## 주요 변경

- `SPEC-027` 1차 안정화 작업을 반영했습니다.
- 런타임 계약과 preflight 경로를 좁은 경계로 분리해 `namba run` 계열 동작을 더 안전하게 다룰 수 있게 했습니다.
- 병렬 실행 lifecycle, 프로젝트 분석, README 렌더링 주변의 구조를 정리하고 회귀 테스트를 보강했습니다.
- 저장소 스캔과 생성 경로의 중복 비용을 줄이기 위한 기반 작업과 측정 산출물을 추가했습니다.
- 일부 명령의 `--help` 처리에서 parse 오류가 잘못 전파되던 회귀를 수정했습니다.

## 운영 메모

- 이번 변경은 사용자 명령 체계를 바꾸지 않는 안정화 중심 릴리스입니다.
- 설치된 CLI는 `namba update`로 최신 릴리스로 올릴 수 있습니다.
