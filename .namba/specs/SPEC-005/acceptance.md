# Acceptance

- [ ] Users can install NambaAI without a local Go toolchain.
- [ ] The default README install path uses release installer scripts, not `go install`.
- [ ] The documented command form is the global `namba` command.
- [ ] Windows installer downloads a release archive and registers the install directory on user PATH.
- [ ] macOS/Linux installer downloads a release archive and registers the install directory on PATH.
- [ ] A GitHub Actions release workflow packages supported platform binaries.
- [ ] Release asset naming is covered by tests.
- [ ] Validation commands pass.
